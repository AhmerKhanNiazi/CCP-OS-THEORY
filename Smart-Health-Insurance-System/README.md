## 🚀 Features
- **Hospital Dashboard:** Submit new claims and track their statuses.
- **Automated Validation:** Basic rule-based risk scoring (Low, Medium, High).
- **Insurance Officer Portal:** Review, Approve, or Reject pending claims.
- **Policyholder Tracking:** Check claim history and status in real-time.

## 🛠️ How to Run Locally

If you have Node.js installed, follow these steps:

1. Open your terminal in this directory (`Smart-Health-Insurance-System`).
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the local development server.
4. Open the link (usually `http://localhost:5173`) in your browser.

Alternatively, you can just run `install.bat` and then `run.bat` located in the main project folder!
